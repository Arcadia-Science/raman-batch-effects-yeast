# About this repo

This repo contains a draft of a short natural-language scientific publication about batch effects in raman spectroscopy.

We are not writing any code in this repo, but rather using it to collaboratively write the publication, assemble figures, etc.

For examples of the kind of content and tone we are aiming for, refer to the following previous publications:

- https://arcadia-science.github.io/2025-raman-spectrometer-comparison/
- https://thestacks.org/publications/dataset-pombe-mutants-srs-cars
- https://thestacks.org/publications/resource-autoopenraman

## Repo structure

The repo is organized as follows:

- `artifacts/`: contains read-only files that are used as references to help us write the publication.
- `figures/`: contains the figures for the publication.
